\subsection{Latihan Membuat \textit{Widget} menggunakan Prinsip SDUI}

Tugas pertama diberikan oleh mentor sebagai latihan pengembangan
\textit{widget} berbasis JSON.\@Proses pengerjaan dilakukan secara
\textit{local} dengan membuat halaman baru khusus untuk latihan (tidak
\textit{push} ke \textit{repository} agar kode tetap bersih). Tersedia
dokumentasi khusus \textit{Front-End} yang dipelajari untuk memahami struktur
dan implementasi pembuatan widget.

\begin{lstlisting}[caption=Contoh JSON untuk Membuat \textit{Widget TabView}, captionpos=b, xleftmargin=2cm, xrightmargin=2cm]
{
  "section_class": 3, 
  "section_orientation": 1,
  "section_main_alignment": 3,
  "section_cross_alignment": 3,
  "section_extra" : {
     "with_navigation_button": true,
     "type" : 1,     
  },
  "value": [
     <List of SECTIONS>    
  ]
};
\end{lstlisting}

Pada tahap awal terdapat kesulitan dalam memahami struktur tersebut, namun
pemahaman meningkat seiring waktu. Melalui proses tersebut, berhasil
dikembangkan beberapa jenis \textit{widget} seperti \textit{Text, TextField,
    Image, Radio, Button, Checkbox, TabView, Dropdown,} dan komponen lainnya.

\subsection{Migrasi Beberapa \textit{Widget} ke Repository FE Terbaru}

Setelah memahami penggunaan JSON untuk membangun UI dengan Flutter, diberikan
tugas \#13228 untuk migrasi beberapa \textit{widget} dari \textit{repository}
sebelumnya (Flutter v3.16.9) ke repository terbaru (Flutter v3.38.5). Selain
itu, diterapkan struktur \textit{Model-View-Controller} (MVC) untuk
meningkatkan keterbacaan (readability) serta skalabilitas (scalability) kode
pada \textit{widget} yang dikembangkan.

Migrasi dilakukan terhadap berbagai \textit{widget}, antara lain
\textit{AntrianTV, TextField, AddImageCamera, Map, Card, CodeSnippet, Calendar,
