\section{Tujuan Magang Industri}

Program magang ini bertujuan memberikan pengalaman praktis bagi mahasiswa, 
khususnya dalam bidang pengembangan front-end di industri healthcare app 
developer. Magang ini juga merupakan bagian dari mata kuliah di semester 
enam yang mencakup persiapan, pelaksanaan, evaluasi, dan publikasi magang 
sebagai bagian dari kurikulum akademik MBKM di FTI Universitas Tarumanagara. 
Beberapa tujuan utama dari program ini sebagai berikut.
\begin{enumerate}[noitemsep]
    \item Memahami peran dan tanggung jawab seorang Front-End Developer dalam 
          ekosistem IT korporasi, terutama dalam menangani isu-isu kritis 
          pada aplikasi kesehatan seperti HealthyOne yang terintegrasi dengan 
          SatuSehat dan BPJS. 
    \item Mengembangkan keterampilan teknis dalam melakukan migrasi sistem 
          dari Flutter v1 ke v2 (menangani tantangan null safety) serta 
          transisi ke React guna meningkatkan performa dan kemudahan 
          pemeliharaan aplikasi berbasis web.
    \item Mengaplikasikan standar penulisan kode tingkat industri melalui 
          implementasi prinsip SOLID dan Clean Code pada komponen React UI 
          yang reusable, guna meminimalisir bug dan mempercepat delivery 
          fitur baru.
    \item Mempelajari dinamika kolaborasi lintas fungsi (QA, Back-End, PM) 
          melalui sistem manajemen tugas OpenProject dan kontrol versi 
          GitLab, dengan target produktivitas yang terukur melalui sistem 
          poin KPI.Melatih adaptabilitas dan profesionalisme dalam lingkungan 
          kerja Full WFO, mengasah kemampuan problem-solving mandiri saat 
          menghadapi bug pada legacy code, serta memperluas jaringan 
          profesional di industri farmasi dan teknologi.
\end{enumerate}
