\section{Struktur Organisasi}

Struktur organisasi di PT Inti Utama Solusindo dirancang untuk mendukung 
kelincahan pengembangan produk digital. Sebagai perusahaan IT, struktur
internalnya sangat fokus pada pembagian peran teknis. Di tingkat eksekutif, 
operasional dipantau oleh pimpinan divisi yang membawahi beberapa manajer 
proyek. Divisi IT di PT IUS terdiri dari beberapa tim fungsional utama:
\begin{enumerate}[noitemsep, topsep=2pt]
    \item Tim Front-End: Bertanggung jawab atas antarmuka pengguna pada 
          platform web dan mobile.
    \item Tim Back-End: Mengelola logika server, integrasi basis data, 
          dan API.
    \item Tim Quality Assurance (QA): Melakukan pengujian menyeluruh untuk 
          menjamin aplikasi bebas dari bug sebelum dirilis.
    \item Tim AI \& Data Science: Mengembangkan algoritma cerdas untuk 
          pengolahan data kesehatan.
    \item Tim UI/UX: Merancang prototipe dan alur pengalaman pengguna.
    \item Tim IT Support: Menangani infrastruktur dan bantuan teknis 
          operasional.
\end{enumerate}

Struktur ini memungkinkan adanya spesialisasi namun tetap menjaga kolaborasi yang erat melalui koordinasi para Project Manager (PM) yang memastikan setiap proyek berjalan sesuai roadmap.
